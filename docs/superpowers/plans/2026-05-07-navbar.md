# Navbar Component Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implementar el componente `Navbar` como barra de navegación fija en la parte inferior para mobile, con 4 ítems (Inicio, Plantas, Recetas, Comunidad), iconos Ionic, y estado activo con fondo circular usando RouterLinkActive.

**Architecture:** Componente standalone que usa `RouterLink` y `RouterLinkActive` nativos de Angular para la navegación y el estado activo. El fondo circular del ítem activo se aplica con CSS puro sobre la clase `active` que añade `RouterLinkActive`. No hay lógica en el componente — solo HTML semántico y SCSS.

**Tech Stack:** Angular 19 (standalone components), `@angular/router` (RouterLink, RouterLinkActive), Ionic Icons (`ion-icon`), Angular Material CSS variables (`--mat-sys-*`), SCSS.

describe('Navbar', () => {

      imports: [Navbar],
      providers: [provideRouter([])]

import { Component, inject, signal } from '@angular/core';
import { RouterOutlet } from '@angular/router';
import { TranslateService } from '@ngx-translate/core';
import { Navbar } from './shared/navbar/navbar';

@Component({
  selector: 'app-root',
  imports: [RouterOutlet, Navbar],
  templateUrl: './app.html',
  styleUrl: './app.scss'
})
export class App {
  protected readonly title = signal('PlantealoApp');

  private translate = inject(TranslateService);

  textoParaTraducir = 'Hola Mundo';

  constructor() {
    this.translate.addLangs(['es', 'ca', 'en']);
    this.translate.setFallbackLang('es');
    this.translate.use('en');
  }
}
